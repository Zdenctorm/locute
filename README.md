# Locute

Soukromé české diktování pro macOS (menu bar, offline WhisperKit).

## Nejnovější verze na tvém Macu

Aktuální kód je na **GitHubu, větev `main`** — ne v posledním DMG z Releases.

```bash
git clone https://github.com/Zdenctorm/locute.git
cd locute
./scripts/install_latest.sh
```

Podrobnosti, úklid duplicitních kopií a Xcode: **[DEVELOPMENT.md](DEVELOPMENT.md)**.

Produkční DMG a auto-update: **[RELEASING.md](RELEASING.md)** · přehled funkcí: **[OVERVIEW.md](OVERVIEW.md)**.
