# Dictator

Soukromé české diktování pro macOS (menu bar, offline WhisperKit).

## Nejnovější verze na tvém Macu

```bash
git clone https://github.com/Zdenctorm/dictator.git
cd dictator
./scripts/install_latest.sh --pull
```
