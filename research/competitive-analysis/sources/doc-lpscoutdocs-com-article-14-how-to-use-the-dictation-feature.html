How to use the Dictation feature

Note that Dictation is only available in MacWhisper sold on www.macwhisper.com, not on the App Store version due to restrictions that Apple puts on apps sold on the Mac App Store.

The dictation feature in MacWhisper lets you transcribe audio into any text field directly. You configure a keyboard shortcut in settings and after that you can just use this shortcut anywhere to quickly type by speaking to your Mac.


How to enable dictation in MacWhisper


You can enable the dictation feature from the home screen by clicking on "Dictation". Or you can enable it from Settings > Dictation. We recommend using the homescreen method as this will also guide you through the setup process.

AI Dictation


You can also use ChatGPT prompts to process the dictated text and immediately perform AI features on it. Some example usecases are:

  • Automatically cleaning up your dictated text by removing spelling and grammar mistakes
  • Translating your dictated text into a different language
  • Expanding on what you said to change an informal tone into a full customer support email

You can create your own dictation prompts in Settings > Dictation.


App Specific Prompts

You can also setup MacWhisper to use different AI prompts depending on the app you're working in.

For more information on how to setup App Specific Prompts go here.


AI Service setup

You will need to have your own AI key setup to use these features, and you need an active MacWhisper Pro license. Upgrade now with 20% off

To use the ChatGPT features you will need to add your own OpenAI API key. That API key also needs to have a valid payment method linked to it. Note that this is not the same as a ChatGPT Plus subscription (it's confusing I know and I've written OpenAI before about this).


Next steps:

  • Make sure you have a valid payment method linked to your account
  • Make sure you have used at least $1 of API costs, which will unlock the GPT 4 access.
